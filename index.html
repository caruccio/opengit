Welcome to Opengit

This script is really simple and as expected does one simple thing:

It will open your git project on github.com from your terminal

Here is how you use it

To open the current branch

$ opengit

To open a specific branch, just name it

$ opengit your-cool-branch-name

It will check if the branch (current or passed as argument) exists on the remote side (github.com).

That's it, really simple and you should try it, you'll love it

And this is how you install it

These instructions are for Linux, OS X users should make the necessary substitutions, if any, I know nothing about OS X.

Create a directory ~/bin

mkdir -p "$HOME/bin"

Add the directory to the beginning of the PATH variable inside .bashrc

echo 'PATH="$HOME/bin:$PATH"' >> "$HOME/.bashrc"

This way, every new instance of Bash will now check for executable scripts in your bin directory.

Now you save the file opengit inside the ~/bin directory and make the file executable with:

chmod +x opengit

If for some reason you don't want/can't grant the permissions, you can run it like this:

bash opengit
